# UI Contract: Provide Partial Results When Some Open Data is Unavailable

**Date**: 2026-03-11
**Spec**: `specs/028-partial-open-data-results/spec.md`

## Views

### Estimate Results

Required elements:
- Missing-factor list and warnings.
- Confidence and completeness indicators.
- High-severity warning when reliability is low.

## Copy Requirements

- Use consistent labels: "Missing data", "Confidence", "Completeness".
- Low-reliability warnings must be explicit.
