# UI Contract: Low/High Range

## Inputs
- UI may request estimate range for a location.

## Results
- Display point estimate and low/high range together with clear uncertainty labeling.
- Show disclaimer that range is not a guaranteed bound.
- If range is unavailable, show point estimate with range-unavailable warning.
- If range adjusted, show range-adjusted warning.

## Accessibility
- Range, warnings, and disclaimers must be visible and accessible via keyboard navigation.
