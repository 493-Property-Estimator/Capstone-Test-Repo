# UI Contract: Top Contributing Factors

## Inputs
- UI allows user to open an explanation panel for an estimate.

## Results
- Show top-N increases and decreases with readable labels, values, and directions.
- Label qualitative explanations when numeric attribution is unavailable.
- Provide map highlight for factors with map context; otherwise show unavailable state.
- Keep estimate visible and provide retry when explanation unavailable.

## Accessibility
- Explanation panel, list items, and warnings must be accessible via keyboard navigation.
