# Research: UC-09 Green Space Coverage Planning

## Decision: Omit green space features on geometry-resolution failure
**Rationale**: Clarification requires omission with explicit absence marking when geometry cannot be resolved.
**Alternatives considered**: Neutral default values (rejected).

## Decision: Count only public/shared-access green spaces
**Rationale**: Clarification requires inclusion only of parks, forests, and recreational fields.
**Alternatives considered**: All land-use green categories (rejected).

## Decision: Use default thresholds/weights when configuration missing
**Rationale**: Exception flow 7a and FR-01-012 require default weighting with logging.
**Alternatives considered**: Fail computation (rejected).
