# Research: UC-15 Top Contributing Factors Planning

## Decision: Configurable top-N factors and contribution format
**Rationale**: Clarifications require configurable top-N count and contribution format (dollars or normalized scores).
**Alternatives considered**: Fixed top-N and fixed dollar impacts (rejected).

## Decision: Only map-visualizable factors support map highlighting
**Rationale**: Clarification requires map highlighting only for factors with map context and explicit unavailable state otherwise.
**Alternatives considered**: Attempting to visualize all factors (rejected).

## Decision: Qualitative explanations when numeric attribution unsupported
**Rationale**: Alternate flow 4a requires qualitative explanation with labeling.
**Alternatives considered**: Hide explanation entirely (rejected).
