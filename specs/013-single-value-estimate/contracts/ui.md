# UI Contract: Single Estimated Value

## Inputs
- UI accepts location input and optional attributes.

## Results
- Display a single estimated value prominently with timestamp and location summary.
- Preserve entered attributes after result display.
- Surface missing-baseline and missing-feature warnings without blocking.
- On validation or normalization errors, highlight inputs and show retry path.

## Accessibility
- Estimated value, warnings, and errors must be visible and accessible via keyboard navigation.
